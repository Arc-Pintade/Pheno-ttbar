This code is an analysis of standard model extansion's consequences on the top quark cross section at CMS.

1) What's in ?

This is a two-part code. The first "creation" part, takes text files of vectors of A matrices as input and calculate average matrices A and put in text files with the number of events implied.
The second "analyze" part takes matrix average text files and create some analysis histograms.
Only the average files are on gitHub. 

2) Launch :

This program is pre-compiled.
To launch the analysis program :
    1) Download the .zip and open a shell
    2) launch the program with : ./analyze
    3) write "analyze" and push "Return" on keyboard
